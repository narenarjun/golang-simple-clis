# motd cli

```bash
go run main.go
```
```bash
Usage:
  motd [flags]

Flags:
  -g, --greeting string   Greeting to use in the message
  -h, --help              help for motd
  -n, --name string       Name to use in the message
  -v, --preview           Preview message instead of writing to /etc/motd   
  -p, --prompt            Prompt for name and greeting
exit status 1
```
